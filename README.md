💰 Expense Tracker App (React)
A modern and intuitive expense tracking application built with React. The app helps users manage their finances efficiently with insightful analytics and a clean, responsive user interface.
________________________________________
Features
• Financial Overview
→ Track expenses on a monthly basis
→ View recent transactions
→ Monitor:
       - Total balance
       - Total expenses
       - Highest spending category
       - Net balance (positive or negative based on income vs expenses)
• Transactions Management
→ Add, view, and manage transactions
→ Admin capabilities:
     - Edit and update transactions
→ User capabilities:
     - View updated transaction data in real time
• Search & Filter
→ Filter transactions by:
     - Date
     - Category
→ Search transactions using description keywords
• Reports & Analytics
→ Monthly financial reports including:
     - Line chart for savings trends
     - Bar graph for income vs expenses
     - Pie chart for category-wise expense distribution
• User Settings
→ Update profile information:
     - Email
     - Preferred currency
→ Manage preferences:
     - Notifications
     - Security settings
• User Experience
→ Toggle between:
     - Dark Mode
     - Light Mode
→ Responsive and user-friendly interface
